# 🌍 AI Disaster Intelligence Platform

An AI-powered Decision Intelligence Platform that helps governments, emergency responders, and citizens prepare for, respond to, and recover from natural disasters. The platform uses Google Gemini AI to analyze disaster reports, images, and environmental data, providing real-time insights, predictions, and recommendations for effective disaster management.

---

## 🚀 Features

- 🤖 AI-powered Disaster Chat Assistant
- 📊 Real-time Disaster Monitoring Dashboard
- 📸 Disaster Image Analysis using Gemini AI
- 📍 Disaster Reporting with Location & Images
- 📈 Disaster Risk Prediction
- 🚨 Real-time Emergency Alerts
- 📄 AI-generated Disaster Reports
- 🔐 Secure User Authentication
- ☁️ Cloud-based Data Storage
- 📱 Responsive User Interface

## 🎯 Problem Statement

Natural disasters such as floods, cyclones, earthquakes, wildfires, and landslides generate large amounts of data from multiple sources, making it difficult for authorities to make quick and informed decisions. This platform leverages Artificial Intelligence to analyze disaster information, predict risks, generate recommendations, and support faster emergency response.

---

## 💡 Solution

The AI Disaster Intelligence Platform combines AI, cloud technologies, and real-time analytics to:

- Monitor disaster situations
- Analyze reports and images
- Predict disaster risks
- Generate AI recommendations
- Support emergency response teams
- Improve public safety and disaster preparedness

## 🔄 Workflow

1. User logs in.
2. Citizen reports a disaster with location and images.
3. Gemini AI analyzes the uploaded data.
4. The platform predicts disaster risks.
5. AI generates recommendations.
6. Dashboard displays alerts and analytics.
7. Authorities make informed decisions.

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Suradahemalatha/AI-disaster-intelligence-platform.git
```

### Navigate to Project

```bash
cd AI-disaster-intelligence-platform
```

### Install Frontend Dependencies

```bash
cd frontend
npm install
npm run dev
```

### Install Backend Dependencies

```bash
cd backend
pip install -r requirements.txt
uvicorn app:app --reload
```

---

## 🔑 Environment Variables

Create a `.env` file and configure:

```env
GEMINI_API_KEY=YOUR_GEMINI_API_KEY
FIREBASE_API_KEY=YOUR_FIREBASE_API_KEY
FIREBASE_PROJECT_ID=YOUR_PROJECT_ID
```
